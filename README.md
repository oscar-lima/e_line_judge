e_line_judge
============

Computer Vision Project Winter Semester 2014

### Clone
    git clone git@github.com:sthoduka/e_line_judge.git

### Build
    cd e_line_judge
    mkdir build
    cd build
    cmake ..
    make

### Run
    ./e_line_judge

or 

    ./e_line_judge file /path/to/video/file
